# Content and SEO

## Goal

Attract people searching for practical local AI GPU buying advice.

The site should rank for high-intent queries and be useful when shared on Reddit, X, LinkedIn, and Discord.

## Positioning

Core message:

> Best value GPUs for local AI.

Supporting angles:

- Local LLM performance, not gaming benchmarks
- VRAM-first buying advice
- Beginner-safe recommendations
- Used GPU value
- Avoid expensive mistakes
- Own your AI

## Tone

Use:

- Opinionated buying-guide tone
- Clear practical verdicts
- Minimal hype
- Light local-AI hacker flavor

Avoid:

- Overly meme-heavy copy
- Fake precision
- Affiliate spam
- Unexplained scores
- Benchmaxxed model fanboying

## Main SEO Targets

Initial keyword themes:

- best GPU for local AI
- best GPU for local LLM
- best GPU for Qwen
- RTX 3090 local LLM
- RTX 4090 local AI
- RTX 5090 local AI
- Arc Pro B70 local LLM
- RX 7900 XTX local LLM
- Tesla P40 local LLM
- best used GPU for AI

## First Comparison Pages

Create after MVP table launch.

Priority pages:

- RTX 3090 vs RTX 4090 for local AI
- RTX 3090 vs RTX 5090 for local AI
- RTX 3090 vs Arc Pro B70
- RTX 3090 vs RX 7900 XTX
- Tesla P40 vs RTX 3090
- RTX 4080 16GB vs RTX 3090
- RTX 4060 Ti 16GB vs RTX 3090

## Page Template

Each comparison page should answer:

- Which is better value?
- Which is better for beginners?
- Which runs the hero model better?
- Which has better software support?
- Which has worse hardware headaches?
- Final verdict

Keep pages short and practical.

## Shareable Content

Useful launch assets:

- GPU tier list image
- “Best GPUs for local AI” chart
- “Why 24GB VRAM matters” post
- “Used RTX 3090 is still the local AI sweet spot” post
- “Cheap server GPUs are not beginner-friendly” post

## Reddit Rules

When sharing on Reddit:

- Be transparent that it is your project
- Ask for corrections
- Mark uncertain data clearly
- Do not overclaim
- Do not spam affiliate links
- Share methodology
- Accept criticism and update data

## Affiliate Rules

Affiliate links are allowed later, but trust comes first.

Rules:

- Clearly disclose affiliate links
- Do not let affiliate payouts affect rankings
- Keep source links visible
- Prefer useful buying links over aggressive CTAs
- Mark suspicious prices

## Trust Content

Every public page should include or link to:

- Methodology
- Last updated date
- Data confidence labels
- Source links
- Caveats

The site should feel like a practical tool, not an SEO farm.
